# Responsive Landing Page

A modern, mobile-first landing page built with HTML5, CSS3, and vanilla JavaScript.

## Features
- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Interactive elements
- Semantic HTML5
- Modern CSS Grid and Flexbox
- Mobile toggle menu
- Back to Top button

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run
Simply open `index.html` in your web browser or use Live Server in VS Code.

## What I Learned

### HTML & Structure
- Semantic HTML5 elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`)
- Proper use of IDs for navigation anchors vs classes for styling
- Structuring content with meaningful class names
- Accessibility attributes (`aria-label`, `rel="noopener noreferrer"`)
- Form structure and input types

### CSS Fundamentals
- **CSS Variables (Custom Properties)**: Creating a design system with reusable values
- **CSS Reset**: Understanding why `box-sizing: border-box` matters
- **Flexbox**: One-dimensional layouts for navigation and forms
- **CSS Grid**: Two-dimensional layouts for feature cards
- **Positioning**: Sticky navigation and fixed back-to-top button

### Responsive Design
- **Mobile-First Approach**: Base styles for mobile, enhanced with media queries
- **Media Queries**: Using `min-width` for progressive enhancement
- **Breakpoints**: 567px (mobile), 768px (tablet), 1024px (desktop)
- **Responsive Typography**: Scaling font sizes across devices
- **Fluid Layouts**: Using percentages, rem units, and max-width

### Design Principles
- **Spacing System**: Consistent 8px-based grid (0.5rem, 1rem, 2rem, 3rem, 4rem)
- **Typography Scale**: Hierarchical font sizing for visual clarity
- **Color Theory**: Primary, secondary, and neutral color usage
- **Hover Effects**: Using `transform` and `box-shadow` for interactivity
- **Transitions**: Smooth animations with `transition: all 0.3s ease`
