## Learning Path

### Level 1: The Foundations (HTML, CSS, JavaScript)
Building blocks of the web - semantic markup, responsive design, and vanilla JavaScript DOM manipulation.

**Status**: In Progress

#### Task 1: Responsive Landing Page
- [x] Project setup complete
- [x] HTML structure implemented (semantic HTML5 with all classes/IDs)
- [x] CSS styling with Flexbox/Grid
- [x] Mobile-responsive design (media queries implemented)
- [x] JavaScript interactivity (mobile menu, back to top, smooth scroll)
- [ ] Testing & refinements
- [ ] Deployed to live server

**What I'll Learn**: Semantic HTML5, Flexbox, CSS Grid, Mobile-First Design, Media Queries
**Key Concepts**: Accessibility, SEO basics, Responsive images
[View Project](./level-1/responsive-landing-page)

#### Task 2: Interactive Form
- [ ] Project setup complete
- [ ] Form HTML structure
- [ ] Real-time validation logic
- [ ] Error handling & feedback
- [ ] Regex patterns for validation
- [ ] Deployed to live server

**What I'll Learn**: Form validation, Regular Expressions, Event listeners, Real-time feedback
**Key Concepts**: User Experience (UX), Input validation patterns
[View Project](./level-1/interactive-form)

## Getting Started

### Prerequisites
```bash
# Check if Node.js is installed (required for Level 2+)
node --version

# Check if Git is installed
git --version
```

### Clone This Repository
```bash
git clone https://github.com/YOUR_USERNAME/Codveda-Internship-Tasks.git
cd Codveda-Internship-Tasks
```

### Running Individual Projects
Each project has its own README with specific instructions. Navigate to the project folder and follow the setup steps.

**Example for Level 1 Projects:**
```bash
cd level-1/responsive-landing-page
# Open index.html in your browser or use Live Server
```

**Example for Level 2+ Projects:**
```bash
cd level-2/spa-react
npm install
npm run dev
```
